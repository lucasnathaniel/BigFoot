# BigFoot

> This project aims to make a fast footprint with nmap, nikto and enum dns

Git: <https://github.com/lucasnathaniel/BigFoot>

## Compilation

```
$ cd bin/ && make && cd ..
```

## Use

```
$ ./bin/BigFoot site.com (Options)
```
### Options

[+] Output:  -o

>Exemple: ./BigFoot google.com -o
